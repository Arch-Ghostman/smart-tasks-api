# smart-tasks-api

API Completa: Nest.js + TypeScript + Jest + Docker + Prisma + PostgreSQL

<ul>
  <li>CRUD completo</li>
  <li>Auth com JWT</li>
  <li>Prisma + PostgreSQL + Docker</li>
  <li>Validação</li>
  <li>Swagger ou Insomnia</li>
  <li>Testes com Jest</li>
  <li>Deploy (Railway/Render)</li>
  
</ul>

## Scripts

- `npm run start:dev`
- `npm run build`
- `npm run test`
- `npm run lint`
- `npm run typecheck`

## Ambiente

Configure `.env.development` com `DATABASE_URL` e `JWT_SECRET`.
